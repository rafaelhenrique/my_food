# library_api

## Installation

Clone this project:

```
git clone https://github.com/rafaelhenrique/my_food.git
```

Enter on directory:

```
cd my_food
```

Create virtualenv:

```
python3.5 -m venv .venv
```

Activate your virtualenv:

```
source .venv/bin/activate
```

Install requirements:

```
pip install -r requirements.txt
```

## Run project

Activate your virtualenv:

```
source .venv/bin/activate
```

Then run:

```
python3.5 app.py
```
